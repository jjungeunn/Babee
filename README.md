**진행 정보**
- 개발기간: 2023.08.01 ~ 2023.09.07
- 진행인원: 4명

- 언어: Java, JSP, HTML/CSS, Javascript

- 프레임워크: Spring Framework 4.10.0 RELEASE, mybatis 1.3.2, ajax, json, bootstrap, jquery 3.6.4
- 데이터베이스: MySQL 8.0
- IDE: Eclipse
- API: 카카오 로그인, 네이버 로그인, 우편번호, 카카오맵, 이메일, 카카오페이, 카카오채널
